Galactic Adventures

This project was about developing the interactive user experience for a created iPad game called Galactic Adventures.

During this process careful consideration was payed attention to how the user will be able to achieve certain goals to navigate through the game and play it. I wanted to create some simple and high fidelity wireframes to illustrate how the experience of using the game would work.
